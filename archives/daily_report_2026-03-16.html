🔥 大模型前沿

Neuron-Aware Data Selection In Instruction Tuning For Large Language Models
👥 Xin Chen, Junchao Wu, Shu Yang
指令调优（Instruction Tuning, IT）已被证明是释放大语言模型（LLMs）强大能力的有效方法。最近的研究表明，过多的IT数据可能会降低LLMs的性能，而仔细选择一小部分高质量的IT数据则可以显著增强其能力。因此，从IT数据集中识别出最有效的子集以有效开发LLMs在特定或通用任务中的能力已成为一个关键挑战。为了解决这一问题，我们提出了一种新颖且高效的框架NAIT。NAIT通过分析IT数据集与目标领域能力之间的神经元激活模式相似性来评估IT数据对LLMs性能的影响。具体来说，NAIT从目标领域能力的领域内数据集中捕获神经元激活模式，以构建可重用和可迁移的神经元激活特征。然后基于候选样本与目标能力预期激活特征之间的相似性进行评估和选择最优样本。实验结果表明，在由NAIT选择的10% Alpaca-GPT4 IT数据子集上训练的表现始终优于依赖外部先进模型或不确定性特征的方法。我们的研究还揭示了神经元激活特征在LLMs不同能力之间的可迁移性。特别是，包含更多逻辑推理和编程特征的IT数据具有很强的通用迁移性，能够使模型在多个任务中发展出更强的能力，而一个稳定的核心数据子集就足以持续激活基本模型能力，并在各种任务中普遍提升性能。 核心贡献：本文提出了NAIT框架，通过分析神经元激活模式的相似性来高效选择高质量的指令调优数据，从而在多种任务中提升大语言模型的性能并揭示了神经元激活特征的可迁移性。

Semantic Invariance in Agentic AI
👥 I. de Zarzà, J. de Curtò, Jordi Cabot
摘要：大型语言模型（LLMs）日益作为自主推理代理，应用于决策支持、科学问题求解以及多智能体协调系统。然而，在关键应用场景中部署LLM代理时，需要确保其在语义等价输入变化下的推理稳定性，我们称之为语义不变性。传统的基准评估方法通过固定的标准问题形式来评估准确性，无法捕捉这一关键的可靠性维度。为解决这一不足，本文提出了一种元形态测试框架，用于系统评估LLM推理代理的鲁棒性。该框架应用了八种保持语义不变的转换方式（包括身份变换、释义、事实重排序、扩展、收缩、学术背景、商业背景和对比式表述），覆盖七个基础模型，涵盖四种不同的架构家族：Hermes（70B, 405B）、Qwen3（30B-A3B, 235B-A22B）、DeepSeek-R1 和 gpt-oss（20B, 120B）。我们的评估涵盖了八个科学领域中的19个分步推理问题。结果显示，模型规模并不能预测鲁棒性：较小的 Qwen3-30B-A3B 模型表现出最高的稳定性（79.6% 的响应具有语义不变性，语义相似度为 0.91），而更大的模型则表现得更为脆弱。 核心贡献：本文提出了一个元形态测试框架，用于系统评估大型语言模型在语义等价输入下推理的鲁棒性，并揭示了模型规模与鲁棒性之间并无直接关系。

Developing and evaluating a chatbot to support maternal health care
👥 Smriti Jha, Vidhi Jain, Jianyu Xu
在资源匮乏的地区，通过基于电话的聊天机器人提供可靠的孕产妇健康信息可以产生重大影响，尤其是在用户健康素养较低且获取医疗服务有限的情况下。然而，部署此类系统在技术上具有挑战性：用户的查询通常简短、描述不充分，并可能混合使用多种语言；回答问题需要结合区域性的具体背景；而症状信息的部分缺失则使得安全的转介决策变得困难。我们介绍了一个由学术研究人员、一家健康科技公司、一个公共卫生非营利组织和一家医院合作开发的印度孕产妇健康聊天机器人。该系统结合了（1）阶段感知的初步评估机制，将高风险查询路由至专家模板；（2）对精心整理的孕产妇/新生儿指南进行混合检索；以及（3）基于大语言模型（LLM）并结合证据的生成。我们的核心贡献是提出了一种在专家监督有限情况下用于高风险部署的评估流程。针对组件级和端到端测试，我们引入了以下内容：（i）一个标注的初步评估基准（N=150），紧急情况召回率达到86.7%，明确报告了漏报与过度升级之间的权衡；（ii）一个合成的多证据检索基准（N=100），包含片段级别的证据标签；（iii）基于真实查询的LLM作为评判者的比较实验（N=781），采用临床医生共同设计的标准；以及（iv）专家验证。研究结果表明，在多语言、嘈杂环境中构建可信的医疗助手，需要多层次的防御性设计和多种方法的评估，而不是依赖单一模型或评估方法。 核心贡献：本文提出了一种在专家监督有限的情况下，适用于高风险场景的多方法评估流程，展示了在多语言、嘈杂环境中构建可信医疗聊天机器人所需的设计与评估策略。

ESG-Bench: Benchmarking Long-Context ESG Reports for Hallucination Mitigation
👥 Siqi Sun, Ben Peng Wu, Mali Jin
随着企业责任日益纳入环境、社会和治理（ESG）标准，ESG报告在许多地区正成为法律要求，并作为记录可持续发展实践和评估公司长期及道德表现的关键渠道。然而，ESG披露内容的长度和复杂性使其难以解释且难以可靠地自动化分析。为了支持可扩展且可信的分析，本文提出了ESG-Bench，这是一个用于评估大型语言模型（LLMs）在理解ESG报告和减少幻觉方面的基准数据集。ESG-Bench包含基于真实ESG报告上下文的人工标注问答对，并带有细粒度标签，表明模型输出是事实支持还是幻觉生成。将ESG报告分析框架化为具有可验证性约束的问答任务，使我们能够系统地评估LLMs在提取和推理ESG内容方面的能力，并提供了一个新用例：在社会敏感和合规关键的场景中减少幻觉。我们设计了特定任务的思维链（CoT）提示策略，并使用带有CoT注释的推理过程对多个最先进的LLMs进行了微调。实验表明，这些基于CoT的方法在减少幻觉方面显著优于标准提示和直接微调，并且这些改进可以推广到ESG领域之外的现有问答基准。 核心贡献：本文提出了ESG-Bench，一个用于评估大型语言模型在处理长文本ESG报告时减少幻觉能力的基准数据集，并展示了基于思维链的方法在提升模型可靠性和泛化性能方面的有效性。

Mending the Holes: Mitigating Reward Hacking in Reinforcement Learning for Multilingual Translation
👥 Yifeng Liu, Siqi Ouyang, Yatish Hosmane Revanasiddappa
大型语言模型（LLMs）在高资源语言对的机器翻译中表现出色，但在低资源语言上的性能仍有不足。现有的后训练方法严重依赖高质量的平行数据，而这些数据对于低资源语言通常稀缺或不可用。本文中，我们提出WALAR，一种仅使用单语文本的强化训练方法，旨在提升LLMs在大量低资源语言上的翻译能力，同时保持其在高资源语言上的表现。我们的核心洞察基于对现有基于源语言的多语言质量评估（QE）模型中失败模式（或“漏洞”）的观察。使用这些QE模型进行强化学习（RL）往往会放大这些漏洞，导致多语言LLMs性能下降。为此，我们在WALAR的强化学习奖励机制中引入了词对齐和语言对齐等技术来缓解这些问题。我们利用WALAR持续训练了一个支持101种语言翻译的LLM。实验表明，我们的新模型在Flores-101数据集的1400个语言方向上显著优于当前最强的开源多语言LLM之一LLaMAX。 核心贡献：提出WALAR，一种通过词对齐和语言对齐缓解质量评估模型缺陷的强化学习方法，在不依赖平行数据的情况下大幅提升多语言LLM在低资源语言上的翻译性能。

Structured Distillation for Personalized Agent Memory: 11x Token Reduction with Retrieval Preservation
👥 Sydney Lewis
Long conversations with an AI agent create a simple problem for one user: the history is useful, but carrying it verbatim is expensive. We study personalized agent memory: one user's conversation history with an agent, distilled into a compact retrieval layer for later search. Each exchange is compressed into a compound object with four fields (exchange_core, specific_context, thematic room_assignments, and regex-extracted files_touched). The searchable distilled text averages 38 tokens per exchange. Applied to 4,182 conversations (14,340 exchanges) from 6 software engineering projects, the method reduces average exchange length from 371 to 38 tokens, yielding 11x compression. We evaluate whether personalized recall survives that compression using 201 recall-oriented queries, 107 configurations spanning 5 pure and 5 cross-layer search modes, and 5 LLM graders (214,519 consensus-graded query-result pairs). The best pure distilled configuration reaches 96% of the best verbatim MRR (0.717 vs 0.745). Results are mechanism-dependent. All 20 vector search configurations remain non-significant after Bonferroni correction, while all 20 BM25 configurations degrade significantly (effect sizes |d|=0.031-0.756). The best cross-layer setup slightly exceeds the best pure verbatim baseline (MRR 0.759). Structured distillation compresses single-user agent memory without uniformly sacrificing retrieval quality. At 1/11 the context cost, thousands of exchanges fit within a single prompt while the verbatim source remains available for drill-down. We release the implementation and analysis pipeline as open-source software.

Deferred is Better: A Framework for Multi-Granularity Deferred Interaction of Heterogeneous Features
👥 Yi Xu, Moyu Zhang, Chaofan Fan
点击率（CTR）预测模型通过建模广泛的特征空间中的交互来估计用户对物品的点击概率。一个基础但常被忽视的挑战是这些特征的内在异质性：它们的稀疏性和信息含量差异极大。例如，像物品ID这样的分类特征极其稀疏，而像物品价格这样的数值特征则相对密集。现有的CTR模型大多忽略了这种异质性，采用统一的特征交互策略，将所有特征同时输入到交互层中。这种方法效果不佳，因为低信息量特征的过早引入会带来大量噪声，并掩盖高信息量特征的信号，从而导致模型崩溃并阻碍鲁棒表示的学习。为了解决上述挑战，我们提出了一种多粒度信息感知延迟交互网络（MGDIN），该网络自适应地延迟将特征引入特征交互过程。MGDIN的核心机制分为两个阶段：首先，它采用多粒度特征分组策略，将原始特征划分为在不同粒度下具有更同质化信息密度的不同组，从而缓解极端个体特征稀疏性的影响，并使模型能够从多个角度捕捉特征之间的交互。其次，通过一种层次化的掩码策略实现延迟交互机制，该策略控制每组特征何时以及如何参与交互——在早期层中屏蔽低信息量的组，并在网络加深的过程中逐步取消屏蔽。这种延迟引入的方式使模型能够在融合其他组较稀疏的信息之前，先基于高信息量特征建立稳健的理解。 核心贡献：本文提出了MGDIN，一种通过多粒度分组和延迟交互机制有效处理特征异质性的CTR预测框架，提升了模型鲁棒性和预测性能。

Bridging Sequential and Contextual Features with a Dual-View of Fine-grained Core-Behaviors and Global Interest-Distribution
👥 Yi Xu, Chaofan Fan, Moyu Zhang
点击率（CTR）预测任务通常通过建模用户行为序列特征和候选物品的上下文特征，来估计用户点击该物品的概率。其中，用户行为序列尤为重要，因为它动态地反映了用户兴趣的实时变化。传统的CTR模型通常将这种动态序列聚合为一个向量后再与上下文特征进行交互。然而，这种方法不仅在聚合过程中导致行为信息的丢失，还严重限制了模型捕捉上下文特征与特定用户行为之间交互的能力，最终影响其对细粒度行为细节的建模能力并降低预测准确性。相反，直接将每个用户行为与上下文特征进行交互的方法虽然保留了更多细节，但计算成本高且引入大量与候选物品无关的行为噪声，这些噪声往往会掩盖那些与候选物品相关的行为信号。为此，我们提出了一种核心行为与分布补偿双视角交互网络（CDNet），从两个互补的角度弥合序列特征与上下文特征之间的交互差距：一是细粒度交互，聚焦于最相关的用户行为与上下文特征；二是粗粒度交互，建模用户整体兴趣分布与上下文特征的关系。通过同时捕捉关键行为细节并保留整体兴趣视角，CDNet有效建模了序列与上下文特征之间的相互作用，而不会带来显著的计算负担。最终，广泛的实验验证了CDNet的有效性。 核心贡献：本文提出CDNet，通过结合细粒度的核心行为交互与粗粒度的兴趣分布建模，高效地融合了用户行为序列与上下文特征，提升了CTR预测的准确性。
