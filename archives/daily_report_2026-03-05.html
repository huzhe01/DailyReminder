🔥 大模型前沿

Pointer-CAD: Unifying B-Rep and Command Sequences via Pointer-based Edges & Faces Selection
👥 Dacheng Qi, Chenyu Wang, Jingwei Xu
构建计算机辅助设计（CAD）模型是一项劳动密集型但对工程和制造至关重要的任务。近年来，大语言模型（LLMs）的发展激发了基于LLM的CAD生成方法，这些方法通过将CAD表示为命令序列来实现自动化建模。然而，在实际场景中，这些方法存在局限性，因为命令序列表示法不支持实体选择（例如面或边），从而限制了其在复杂编辑操作（如倒角或圆角）中的应用能力。此外，在草图和拉伸等连续变量离散化过程中可能会导致拓扑错误。为了解决这些问题，我们提出了Pointer-CAD，一种新颖的基于LLM的CAD生成框架，该框架利用基于指针的命令序列表示法，将B-Rep模型的几何信息显式地整合到序列建模中。具体而言，Pointer-CAD将CAD模型生成分解为多个步骤，并根据文本描述以及之前步骤生成的B-Rep进行后续步骤的生成。当某个操作需要选择特定几何实体时，LLM会预测一个指针，从可用集合中选择最符合特征一致性的候选对象。这种选择操作还能减少基于命令序列表示法中的量化误差。为了支持Pointer-CAD的训练，我们开发了一条数据标注流水线，用于生成专家级的自然语言描述，并据此构建了一个包含约575,000个CAD模型的数据集。大量实验结果表明，Pointer-CAD能够有效生成复杂的几何结构，并将分割误差降低到极低水平，显著优于以往的命令序列方法，从而大大缓解了由量化误差引起的拓扑不准确性。 核心贡献：Pointer-CAD提出了一种结合B-Rep模型几何信息与命令序列表示的新型CAD生成框架，通过引入指针机制提升复杂几何结构的生成能力和减少拓扑错误。

AILS-NTUA at SemEval-2026 Task 12: Graph-Based Retrieval and Reflective Prompting for Abductive Event Reasoning
👥 Nikolas Karafyllis, Maria Lymperaiou, Giorgos Filandrianos
我们为SemEval 2026任务12：反溯事件推理提出了一种获奖的三阶段系统，该系统结合了基于图的检索、通过反思性提示进化优化的提示设计驱动的大型语言模型（LLM）反溯推理以及事后一致性强制机制；我们的系统在评估阶段排行榜中排名第一，准确率达到0.95。对14个模型（7个家族）的跨模型错误分析揭示了三种共享的归纳偏置：因果链不完整、近因偏好和显著性偏差，这些偏差在不同模型家族中的收敛（减少了51%的原因数量）表明多标签因果推理中的失败模式是系统性的而非特定于某个模型的。 核心贡献：本文提出了一种在反溯事件推理任务中表现优异的三阶段系统，并通过跨模型分析揭示了多标签因果推理中普遍存在的系统性偏差。

World Properties without World Models: Recovering Spatial and Temporal Structure from Co-occurrence Statistics in Static Word Embeddings
👥 Elan Barenholtz
最近的研究将从大型语言模型（LLM）隐藏状态中线性可恢复的地理和时间变量解释为具有类似世界内部表示的证据。我们测试了一个更简单的可能性：即相关的结构大部分已经隐含在文本本身之中。通过将相同类别的岭回归探测器应用于基于共现的静态词嵌入（GloVe 和 Word2Vec），我们发现可以显著恢复地理信号，且时间信号较弱但可靠，城市坐标的保留测试 R² 值为 0.71-0.87，历史出生年份为 0.48-0.52。语义邻近分析和目标子空间消融表明，这些信号强烈依赖于可解释的词汇梯度，尤其是国家名称和与气候相关的词汇。这些发现表明，普通的词共现比通常认为的更能保留丰富的空间、时间和环境结构，揭示了简单的静态嵌入仅从文本中就能保留世界结构的显著而被低估的能力。因此，线性探测的可恢复性本身并不能证明超越文本的表征转变。 核心贡献：本文证明静态词嵌入中的共现信息能够捕捉到丰富的地理和时间结构，表明线性探测方法所揭示的世界知识可能更多来自文本本身而非模型的复杂内部表征。

The Company You Keep: How LLMs Respond to Dark Triad Traits
👥 Zeyi Lu, Angelica Henestrosa, Pavel Chizhov
大型语言模型（LLMs）通常表现出高度顺从和强化的对话风格，也被称为AI谄媚行为。尽管这种行为在一定程度上是被鼓励的，但在面对反映负面社会倾向的用户提示时可能会引发问题。此类回应可能反而加剧有害行为，而非减轻其影响。在这项研究中，我们使用一个精心整理的数据集，分析了LLMs对表达不同水平黑暗三联征特质（马基雅维利主义、自恋和反社会人格）的用户提示的反应。我们的分析发现，所有模型主要表现出纠正性行为，但在某些情况下也会出现强化性的输出。模型的行为还取决于负面特质的严重程度，并在回应的情感倾向上有所不同。我们的研究结果对设计更安全的对话系统具有启示意义，这些系统能够在用户从良性请求转向有害请求时进行检测并作出适当的回应。 核心贡献：该研究揭示了LLMs在面对具有黑暗三联征特质的用户输入时的行为模式，并为构建能够识别和适当应对有害请求的安全对话系统提供了重要见解。

Position: Vector Prompt Interfaces Should Be Exposed to Enable Customization of Large Language Models
👥 Liangwei Yang, Shiyu Wang, Haolin Chen
随着大型语言模型（LLMs）从研究原型转变为现实世界中的系统，定制化已成为一个关键瓶颈。尽管文本提示已经能够定制LLM的行为，我们认为仅使用文本的提示方式并不适合作为可扩展、稳定且仅用于推理的定制接口。本文主张模型提供者应将**向量提示输入**作为LLM定制的公共接口的一部分。我们通过诊断证据支持这一观点，表明向量提示调优在监督增加时仍能持续改进，而基于文本的提示优化则很快趋于饱和；此外，向量提示表现出密集的全局注意力模式，表明其具有不同的控制机制。我们进一步讨论了在现实部署约束下为何仅推理的定制变得越来越重要，并解释为何在标准黑盒威胁模型下公开向量提示不会显著增加模型泄露的风险。最后，我们呼吁社区重新思考提示接口作为LLM定制核心组件的重要性。 核心贡献：本文提出应将向量提示作为LLM定制的标准接口，以实现更高效、稳定的模型行为控制，并探讨其在实际部署中的可行性和安全性。

Causality Elicitation from Large Language Models
👥 Takashi Kameyama, Masahiro Kato, Yasuko Hio
Large language models (LLMs) are trained on enormous amounts of data and encode knowledge in their parameters. We propose a pipeline to elicit causal relationships from LLMs. Specifically, (i) we sample many documents from LLMs on a given topic, (ii) we extract an event list from from each document, (iii) we group events that appear across documents into canonical events, (iv) we construct a binary indicator vector for each document over canonical events, and (v) we estimate candidate causal graphs using causal discovery methods. Our approach does not guarantee real-world causality. Rather, it provides a framework for presenting the set of causal hypotheses that LLMs can plausibly assume, as an inspectable set of variables and candidate graphs.

📊 广告与推荐算法

IDProxy: Cold-Start CTR Prediction for Ads and Recommendation at Xiaohongshu with Multimodal LLMs
👥 Yubin Zhang, Haiming Xu, Guillaume Salha-Galvan
在广告和推荐系统中，点击率（CTR）模型严重依赖于项目ID嵌入，但在新项目冷启动场景下表现不佳。我们提出了IDProxy，一种利用多模态大语言模型（MLLMs）从丰富的内容信号中生成代理嵌入的解决方案，从而在没有使用数据的情况下实现对新项目的有效CTR预测。这些代理嵌入与现有的ID嵌入空间显式对齐，并与排序模型一起在CTR目标下端到端优化，使其能够无缝集成到现有的大规模排序流水线中。离线实验和在线A/B测试验证了IDProxy的有效性，该方法已成功部署在小红书“探索流”的内容推送和展示广告功能中，每天服务数亿用户。 核心贡献：提出IDProxy，通过多模态大语言模型生成代理嵌入，解决了广告和推荐系统中新项目冷启动下的CTR预测问题，并实现了大规模系统的高效集成与应用。

MAC: A Conversion Rate Prediction Benchmark Featuring Labels Under Multiple Attribution Mechanisms
👥 Jinqi Wu, Sishuo Chen, Zhangming Chan
多归属学习（MAL）是一种有前景的转化率（CVR）预测学习范式，它通过从多种归属机制生成的转化标签中学习来提升模型性能。然而，目前公开的CVR数据集中转化标签仅由单一归属机制生成，这限制了MAL方法的发展。为了解决这一数据缺口，我们构建了首个包含多种归属机制下标签的公共CVR基准数据集——多归属基准（MAC）。此外，为了促进MAL领域的可复现研究，我们开发了PyMAL，一个涵盖广泛基线方法的开源库。我们在MAC上进行了全面的实验分析，并揭示了三个关键发现：(1) MAL在不同归属设置下都能带来稳定的性能提升，特别是在用户具有长转化路径的情况下效果更显著；(2) 在大多数情况下，性能提升随着目标复杂度的增加而扩大；但当预测首次点击转化目标时，简单地添加辅助目标反而有害，这突显了精心选择辅助目标的重要性；(3) 两个架构设计原则至关重要：一是充分学习多归属知识，二是充分利用这些知识服务于主任务。基于这些发现，我们提出了不对称专家混合（MoAE），一种结合多归属知识学习和主任务导向知识利用的有效MAL方法。在MAC上的实验表明，MoAE显著超越了现有的最先进的MAL方法。我们相信，我们的基准和见解将推动MAL领域的未来研究。MAC基准和PyMAL算法库已公开在 https://github.com/alimama-tech/PyMAL。 核心贡献：本文提出了首个支持多归属学习（MAL）的公开CVR基准数据集MAC和开源库PyMAL，并基于此提出了一种新的MAL方法MoAE，在多个实验场景下均取得显著性能提升。
