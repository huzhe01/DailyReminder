HPRO: Hierarchical Progressive Reward Optimization via Preference Extraction for Emotional Text-to-Speech
👥 Sihang Nie, Xiaofen Xing, Rui Xing
最近，基于大语言模型（LLM）的文本到语音（TTS）系统在自然度方面取得了显著进展。然而，标准的监督微调方法通常会收敛到统计意义上的平均韵律，从而限制了情感表达的表现力。虽然偏好驱动的优化提供了一种有前景的替代方案，但现有方法面临两个结构性不匹配问题：信息冲突，即共享潜在空间中的内容和情感会产生冲突的梯度，导致奖励劫持和语义退化；以及尺度差距，即稀疏的句子级奖励难以指导密集的帧级生成。为了解决这些问题，我们提出了HPRO——一种层次化的渐进式奖励优化框架。在HPRO中，我们引入了HD-Emo编解码器作为新的可微分奖励模型，用于解决信息冲突问题。该模型将语音分解为独立的内容和风格偏好标记，在结构上将情感优化与语义内容分离。在此基础上，HPRO通过逐步对齐帧级、词级和句子级目标来弥合尺度差距。实验表明，HPRO显著提升了情感表现力，同时有效保留了语言的可理解性。代码和音频样本可在 https://xxh333.github.io/hpro-demo/ 获取。 核心贡献：提出HPRO框架，通过结构化的情感与语义分离及多粒度目标对齐，有效提升情感TTS系统的表达能力并避免语义退化。

Cognitive Episodes in LLM Reasoning Traces Enable Interpretable Human Item Difficulty Prediction
👥 Chenguang Wang, Ming Li, Xinyue Zeng
预测人类题目难度是教育评估中的核心问题，其中可靠的估计支持公平性和有效的测验构建。现有方法通常依赖于成本高昂的人类校准或题目标级别文本表示，提供的关于使题目变难的认知过程的证据有限。我们认为，难度不应仅被视为题目文本的属性，也应被视为题目所引发的问题解决负担的可观测后果。大型推理模型（LRMs）通过推理轨迹提供了可扩展的过程证据，但这些证据必须结构化以支持可解释的建模。为此，我们引入了 Epi2Diff（Episode to Difficulty），一种将 LRM 推理轨迹映射为基于认知的事件序列的框架。这些事件将轨迹片段分组为功能性的问题解决状态，使得可以通过推理规模、努力分配和状态转换来建模难度。Epi2Diff 提取紧凑的事件动态特征，并将其与语义题目表示相结合，用于人类难度预测。在四个真实世界的人类难度数据集上的实验表明，Epi2Diff 一致优于强大的基线方法，包括微调的小型语言模型、LLM 上下文学习和监督式 LLM 适配。在 SAT 衍生的分类基准上，Epi2Diff 相较于监督式 LLM 微调基线平均相对提升了 8.1%。进一步分析显示，更难的题目会引发更多需要努力、迭代和以实现为中心的事件动态，而不仅仅是更长的回答。这些结果表明，LRM 推理轨迹中的认知事件提供了一种具有预测性和可解释性的人类题目难度过程表示，为使用推理模型进行教育测量提供了新的视角。 核心贡献：本文提出 Epi2Diff 框架，利用 LRM 的推理轨迹中的认知事件动态来建模和预测人类题目难度，实现了比现有方法更高的预测性能和可解释性。

From Tokens to States: LLMs as a Special Case of World Models and the Continuous Path Beyond
👥 Paul Dubois
摘要：人工智能领域将大语言模型（LLMs）与世界模型之间的关系视为二元对立：LLMs预测标记；而世界模型模拟现实。2022年，Yann LeCun提出，实现通用智能需要放弃自回归的标记预测，转而采用潜在空间架构。这种观点过于二元化。本文提出了两个主张。首先，LLMs是世界模型的一种退化的特例：其状态空间是所有标记序列的集合，唯一的行为是添加一个标记，因此世界模型是对LLMs的严格泛化，而非替代。其次，从NTP到JEPA之间存在一个自然的连续光谱，中间阶段包括多标记预测、未来摘要预测和下一个潜在状态预测，这些方面已有当前研究涉足。沿着这一光谱移动，意味着逐步放松LLMs的限制条件。同时，这也逐渐放弃了使LLMs能够大规模训练的两个实际优势：互联网规模的自监督数据以及专为离散标记预测设计的Transformer架构。这两个问题被作为开放的研究课题进行探讨：数据问题（从自监督文本到工具化动作标注环境的陡峭悬崖），以及架构问题（Transformer是否能推广到连续状态预测，或者是否需要新的基本组件）。 核心贡献：本文重新定义了大语言模型与世界模型的关系，指出LLMs是世界模型的一个特例，并提出两者之间存在连续演进路径，从而为通向更通用智能的模型发展提供了理论框架和研究方向。

Mechanism-Driven Monitors for Preemptive Detection of LLM Training Instability
👥 Ruixuan Huang, Yipei Wang, Wenyi Fang
前沿的大语言模型训练需要消耗大量的加速器资源和长时间的计算，因此当稳定性故障发生时，其代价极高。在数值或超参数错误已经破坏了训练动态之后，这种不稳定可能持续数千步，而损失值和梯度范数仍看起来正常。我们研究了基于机制驱动的训练不稳定性检测方法，通过从每个关键模块的功能作用以及最早可预期失败产生可测量特征的计算位置推导出内部监控指标。对于低精度的flash attention，我们监测QK双线性分解的谱熵，在损失完全崩溃之前其一阶项就会出现异常。对于MoE路由器，我们则根据其在专家选择中的角色推导出相关指标。我们在低精度注意力、大学习率以及组合故障上的故障注入实验表明，这些信号为不同类型的故障提供了独特的特征，并能在损失发散前数千步就触发预警。 核心贡献：本文提出了一种机制驱动的训练不稳定性检测方法，通过设计针对关键模块的早期监控指标，在损失发散之前数千步就能识别多种训练故障。

Scaling limit of the Random Language Model
👥 Eric De Giuli
我们发展了随机语言模型（RLM）的定量理论，该模型是一个随机上下文无关文法的集合。在一种标度极限下，隐藏符号的数量 $N \to \infty$，而文法温度 $\tildeε_d \to 0$，但保持 $x = \tildeε_d \log N$ 固定。在这个极限下，模型可以通过规则使用模式上的大偏差原理进行可控描述。一个半退火近似将问题映射到一类具有非平凡组合结构的随机能量模型中。我们证明，当 $x_c=1/8$ 时，RLM 出现凝聚相变，在此以下规则使用集中化，语言统计表现出对语料长度的非平凡依赖。另一个特征尺度 $x=1/2$ 标志着熵从最大值开始减少。在这些不同区域中，我们推导出关于不同规则数量、熵及相关可观测量的显式标度律，并识别出由文法规模、语料长度和温度相互作用控制的不同标度、饱和和临界区域。 核心贡献：本文通过引入大偏差原理和半退火近似，系统分析了随机语言模型在不同标度极限下的统计行为，并揭示了其在特定临界点处的语言相变现象及普遍统计性质。

Single and Multi Truth Data Fusion using Large Language Models
👥 Hira Beril Kucuk, Norman W Paton, Jiaoyan Chen
Data fusion, also known as truth discovery, is a data integration problem that aims to determine the correct value or set of values for each attribute of an object when presented with potentially conflicting values from multiple sources. Data fusion tasks belong to two main categories: single-truth scenarios, where each attribute has only one correct value, and multi-truth scenarios, where multiple values can be valid simultaneously. This paper investigates the use of Large Language Models (LLMs) in data fusion tasks for tabular data. Various prompting strategies, encompassing both single-truth and multi-truth scenarios, are investigated empirically. Domain-dependent, domain-independent, zero-shot and one-shot prompts are evaluated on three different benchmark datasets. Experimental results demonstrate that LLM-based approaches outperform traditional unsupervised truth discovery methods, such as DART and LTM, across all datasets. The codebase of this study has been made publicly available on GitHub.
