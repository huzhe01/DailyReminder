🔥 大模型前沿

Context-Aware RL for Agentic and Multimodal LLMs
👥 Peiyang Xu, Bangzheng Li, Sijia Liu
大型语言模型（LLMs）在回答需要从长或复杂上下文中识别出小但关键的证据时经常失败，例如工具轨迹中的一行代码或图像中的细微细节。我们提出了ContextRL，这是一种上下文感知的强化学习（RL）方法，通过一个**间接**的辅助目标来提升长期推理和多模态性能。与仅监督最终答案不同，ContextRL向模型提供一个查询、一个答案以及两个高度相似的上下文，并奖励模型选择支持该查询-答案对的上下文，从而鼓励细粒度的上下文理解。我们在两个领域构建了对比上下文数据：对于代码代理，轨迹作为上下文，通过条件过滤生成1000对；对于多模态推理，图像作为上下文，通过生成编辑和相似性搜索生成7000对。ContextRL在5个长期基准测试中平均提升了标准GRPO +2.2%，在12个多样化的视觉问答基准上平均提升了+1.8%。为了分离所提出目标的效果与额外数据的影响，我们将其与使用相同对比上下文的标准查询-上下文-答案示例的数据增强基线进行比较。这些基线几乎没有带来改进，表明提升来自于所提出的上下文选择目标，而非对比数据本身。 核心贡献： 本文提出了ContextRL，一种通过上下文选择目标显著提升大语言模型在长期推理和多模态任务中的表现的强化学习方法。

Benchmarking LLM Agents on Meta-Analysis Articles from Nature Portfolio
👥 Anzhe Xie, Weihang Su, Yujia Zhou
元分析是一种要求较高的证据综合方法，它结合了文献检索、基于PI/ECO的纳入研究筛选和统计聚合。其结构化、可验证的工作流程使其成为评估系统性科学推理的理想基础，但现有基准测试在从检索到筛选再到综合的全流程中缺乏真实标注数据。我们引入了MetaSyn，这是一个包含442项由专家策划的元分析的数据集，来源于自然出版集团的期刊。每个条目都配有一个研究问题与PI/ECO标准、一个包含140,000篇PubMed文章的检索语料库、经过验证的阳性研究、在主题上相似但不符合PI/ECO标准的难例（hard negatives），以及完整的搜索策略和时间范围。 对十二种流水线配置（九种RAG变体和一种协议驱动代理）进行基准测试揭示了一个关键的筛选瓶颈：尽管在K=200时检索的召回率上限达到90.9%，但没有任何系统能恢复超过52.7%的真实纳入文献。当前的大语言模型无法可靠地区分在主题相关性相近的研究池中符合PI/ECO标准的研究与不符合的标准干扰项。阶段归因指标能够捕捉系统成功与失败的具体环节；单一端到端评分无法做到这一点。 核心贡献：本文提出了MetaSyn——首个全面涵盖元分析全流程的基准数据集，并揭示了当前大语言模型在文献筛选阶段的关键性能瓶颈。

KVEraser: Learning to Steer KV Cache for Efficient Localized Context Erasing
👥 Mufei Li, Shikun Liu, Dongqi Fu
在KV缓存上进行后处理上下文擦除具有挑战性，因为局部编辑会产生全局影响：一旦某个片段被处理，其影响会传播到后续所有标记的缓存状态中。这一问题在长上下文大语言模型（LLM）应用中自然出现，例如过时检索的事实、错误的工具观察、撤销的用户偏好或有害的提示注入，这些可能只有在预填充之后才能被识别。精确擦除必须重新计算被删除片段之后的所有标记，使其计算成本取决于后缀长度而非被擦除片段的长度。我们提出了KVEraser，这是一种学习型的KV缓存编辑方法，用于高效实现局部上下文擦除。给定一个已处理的上下文和一个要移除的片段，KVEraser仅用学习到的引导状态替换被擦除区间的KV状态，同时保留其余缓存不变。为了学习可迁移的擦除机制，我们构建了一个两阶段训练流程：通用片段邻域预训练教会擦除器抑制被擦除片段的影响，而任务特定微调则将这种能力适配到下游场景。实验表明，在1K-32K上下文长度的领域内任务中，KVEraser的擦除后性能几乎与完全重新计算相当，但其延迟仅增加24%，而完全重新计算的延迟增加了17.6倍。KVEraser还能够泛化到未见过的长文档问答任务中，其中包含有害的事实干扰项，在近似基线中表现最佳，并比完全重新计算快3-4倍。 核心贡献：KVEraser提出了一种高效且可泛化的KV缓存编辑方法，能够在不显著增加计算开销的前提下，实现对长上下文中特定片段的精准擦除。

DEEPRUBRIC: Evidence-Tree Rubric Supervision for Efficient Reinforcement Learning of Deep Research Agents
👥 Minghang Zhu, Chuyang Wei, Junhao Xu
深度研究智能体通过检索并推理相关证据来生成长篇报告。基于评分标准的强化学习（rubric-based reinforcement learning）通过将报告质量转化为可检查的标准作为奖励信号，从而优化这些智能体，但其效率取决于这些标准是否可靠地捕捉了任务范围和证据需求。大多数现有研究依赖大型语言模型（LLM）为给定查询生成评分标准，但当模型无法准确推断潜在的信息需求时，生成的评分标准可能不完整，从而降低强化学习的效率。为获得更可靠的查询—评分标准监督数据，我们提出了 DeepRubric，这是一个数据构建框架，它反转了这一过程：与其为给定查询推断评估标准，DeepRubric 首先确定一个基于证据的报告应被评估的内容，然后从这些评估目标中合成对齐的查询—评分标准对。从一个采样的种子主题出发，DeepRubric 通过递归扩展基于证据的子问题构建一个证据树，其叶子节点作为原子且可验证的评估目标。随后，它利用该证据树合成训练用的查询和评分标准，确保奖励机制精确评估查询所请求的信息。使用 DeepRubric，我们构建了9000个查询—评分标准监督示例，并通过基于评分标准的 GRPO 训练了 DeepRubric-8B，在三个基准测试中，其性能与先前最先进的开源深度研究模型相当，但所需的 RL GPU 小时数减少了约13倍。 核心贡献：提出 DeepRubric 框架，通过逆向构建证据树和对齐的查询—评分标准对，显著提升基于评分标准的强化学习在训练深度研究智能体中的效率和可靠性。

TokenPilot: Cache-Efficient Context Management for LLM Agents
👥 Buqiang Xu, Zirui Xue, Dianmou Chen
随着大语言模型（LLM）代理在长周期会话中的部署，上下文的累积显著增加了推理成本。现有的方法通常利用文本修剪或动态内存驱逐来最小化令牌足迹；然而，这些方法不受约束的序列修改会改变布局，导致前缀不匹配和缓存失效。这揭示了文本稀疏性与提示缓存连续性之间的重要权衡。为了解决这一问题，我们提出了 TokenPilot，一个双粒度的上下文管理框架。全局层面，Ingestion-Aware Compaction 作为框架的核心机制，用于稳定提示前缀并在输入阶段消除开放世界环境噪声。局部层面，Lifecycle-Aware Eviction 监控上下文片段的剩余效用，采用保守的批量调度策略，在任务相关性过期时才卸载内容片段。在 PinchBench 和 Claw-Eval 基准上的实验表明，在隔离模式下 TokenPilot 将成本分别降低了 61% 和 56%，而在连续模式下则分别降低了 61% 和 87%，同时保持了与现有系统相当的性能表现。TokenPilot 已被集成到 LightMem2 中，代码可在 https://github.com/zjunlp/LightMem2 获取。 核心贡献：提出了一种名为 TokenPilot 的双粒度上下文管理框架，在降低 LLM 推理成本的同时维持了较高的性能表现。

Exploring Extrinsic and Intrinsic Properties for Effective Reasoning with Code Interpreter
👥 Patomporn Payoungkhamdee, Napat Laosaengpha, Jenta Wonglertsakul
Reasoning with a Code Interpreter (CI) has emerged as an effective paradigm for enhancing the reasoning capabilities of large language models (LLMs) through executable computation and iterative verification. Despite its growing adoption, the behavioral properties underlying effective code reasoning remain largely underexplored. In this work, we investigate code reasoning from two distinct perspectives inspired by prior studies of natural language reasoning: extrinsic properties, represented by crucial tokens, and intrinsic properties, represented by code-specific cognitive behaviors. Across multiple LLMs, we find that stronger CI reasoning models consistently exhibit a higher prevalence of crucial tokens and cognitive behaviors, particularly verification, backtracking, and backward chaining. Building on these observations, we examine how these properties can be leveraged during both inference and training. At inference time, appending code-specific crucial tokens improves performance on several reasoning capabilities, including mathematical, ordering, and optimization, while yielding limited benefits elsewhere. At training time, augmenting a state-of-the-art framework with code-specific cognitive behaviors improves supervised fine-tuning and reinforcement learning performance in two of three evaluated models. Further analysis shows that these behaviors reduce overthinking in incorrect responses and improve token efficiency, while also revealing factors that limit gains in a certain model. Our findings provide the first systematic characterization of effective reasoning with CI and demonstrate both the potential and limitations of leveraging key properties to improve CI-based reasoning.

📊 广告与推荐算法

Beyond Positive Signals: Unlocking Implicit Negative Behaviors for Enhanced Sequential User Modeling
👥 Zexuan Cheng, Yue Liu, Jun Zhang
用户行为序列建模已成为现代点击率（CTR）预测的核心组成部分。近年来，研究界投入了大量精力改进序列的编码方式，从目标感知注意力机制、兴趣演化网络到统一架构以联合处理序列和非序列特征。然而，一个更为基础的问题仍未得到充分探索：行为序列应包含哪些内容？目前的做法仅从正向交互（点击、购买、完成）中构建序列，而数量远为丰富的隐式负向行为（跳过、低参与度、滑动通过）则被严重低估。随着正向序列长度带来的增益趋于边际递减，我们重新审视这一未充分利用的数据源，并将其纳入序列建模框架。在本文中，我们证明，在固定长度预算下，将正负行为标记按时间顺序交错构成混合极性行为序列，能一致优于仅使用正向序列的方法，且在多种模型架构下计算开销几乎可以忽略不计。我们进一步识别出一种朴素极性嵌入所固有的语义不可区分问题，并提出了一种轻量级的目标条件门控机制——目标感知极性融合（TAPF），通过区分行为证据来实现额外提升。值得注意的是，即使简单的极性偏置基线也能捕获大部分性能提升，这表明主要贡献在于混合极性数据范式的本身。我们在三个公开基准上的实验结果表明，该方法在五种架构下的相对AUC指标提升了+1.9%到+9.6%，验证了该方法的实际价值。 核心贡献：本文首次系统地引入并验证了混合极性行为序列在用户建模中的有效性，提出了新的建模方法并展示了其在多个基准任务中的显著性能提升。
