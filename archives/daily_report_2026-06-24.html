L3Cube-MahaPOS: A Marathi Part-of-Speech Tagging Dataset and BERT Models
👥 Hariom Ingle, Ronit Ghode, Ishwari Gondkar
词性标注（POS）是支撑机器翻译、信息抽取和句法分析等任务的基础自然语言处理任务。尽管马拉地语拥有超过8300万使用者，并在全球最常用的语言中排名前二十，但其在标注语料库和标准化评估基准方面资源严重不足。马拉地语由于丰富的形态变化、相对自由的词序、缺乏大小写规范以及与印地语和英语的广泛混合使用，给计算建模带来了独特的挑战。我们提出了 L3Cube-MahaPOS，一个高质量的马拉地语词性标注数据集，包含从新闻文本中手工标注的32,354个句子。该数据集由精通马拉地语的标注人员团队按照16个标签的通用依赖（Universal Dependencies）标准进行完全手动标注。结构化的预处理流程包括Unicode规范化、梵文字符识别的分词和噪声过滤，以确保所有数据分割中的标签一致性。我们在HMM、CRF、BiLSTM、BiLSTM+CharCNN、MuRIL以及马拉地语专用的MahaBERT-v2六类模型上对该数据集进行了基准测试。最佳系统在15个评估标签类别上的词级准确率达到88.67%，宏F1得分为81.67%。我们公开了该数据集、标注指南和训练好的模型检查点，以促进马拉地语自然语言处理领域的进一步研究。 核心贡献：本文提出了首个大规模、高质量的马拉地语词性标注数据集 L3Cube-MahaPOS，并提供了多种模型的基准结果，推动了马拉地语NLP的研究发展。

SHERLOC: Structured Diagnostic Localization for Code Repair Agents
👥 Hovhannes Tamoyan, Sean Narenthiran, Erik Arakelyan
LLM代理通过多轮工具使用来解决仓库级别的编程任务，但它们将一半的预算用于编辑之前的故障定位。尽管已经出现了一些专门的定位框架，但这些框架仍被评估为文件检索，而不是可操作的诊断，因此生成的位置缺乏修复代理所需的诊断上下文。我们引入了SHERLOC（基于结构化假设驱动的探索与推理的定位框架），这是一个无需训练的框架，它结合了一个推理LLM、紧凑的仓库工具和自我恢复机制，无需微调或多方代理协作。SHERLOC在不同模型规模下实现了最先进的定位效果：在SWE-Bench Lite上的准确率@1为84.33%，在SWE-Bench Verified上的召回率@1为81.27%；在约30B参数的情况下，其表现可媲美甚至优于其他代理方法。将我们的定位结果和诊断发现注入修复代理后，在SWE-Bench Verified上平均提升了5.95个百分点的解决率，同时将定位和总token数量分别减少了36.7%和23.1%。 核心贡献：SHERLOC提出了一种无需训练的代码故障定位框架，显著提升了定位准确性并有效降低了计算开销，从而增强了代码修复代理的整体性能。

Are We Ready For An Agent-Native Memory System?
👥 Wei Zhou, Xuanhe Zhou, Shaokun Han
摘要：大型语言模型（LLM）代理的内存系统已迅速从简单的检索增强机制发展为支持持久信息存储、检索、更新、整合以及动态生命周期管理的数据管理系统。尽管如此，现有的评估方法仍主要通过端到端任务成功指标（如F1、BLEU）来衡量代理内存性能，并将底层系统视为一个整体黑盒。因此，关键的系统级问题，如操作成本、内存模块间的架构权衡以及在动态知识更新下的鲁棒性，仍未得到充分研究。本文从数据管理的角度对代理内存进行了系统的实验研究。我们提出了一个分析框架，将代理内存分解为四个核心模块：内存表示与存储、提取、检索与路由以及维护。基于该框架，我们在五个基准工作负载（覆盖11个数据集）下评估了12种代表性内存系统和两个参考基线。我们的广泛端到端评估表明，没有任何一种架构能在所有场景中占优；相反，其有效性高度依赖于内存结构与工作负载瓶颈的一致性。此外，通过细粒度消融实验，我们量化了各个模块对表示保真度、检索精度、更新正确性和长期稳定性的影响。最后，我们在真实工作负载下揭示了成本与性能之间的权衡关系，显示局部维护比全局重组更具成本效益。基于这些发现，我们确定了构建真正面向代理的内存系统的有前景方向。 核心贡献：本文提出了一种系统性的代理内存评估框架，揭示了不同内存架构在多种工作负载下的表现差异，并通过详尽实验量化了各模块的关键影响，为设计高效、成本可控的代理原生内存系统提供了理论依据和实践指导。

CANDLE: Character-level Arabic Noise Deduplication using Lightweight Encoder
👥 Faris Alasmary, Taif Nono, Orjuwan Zaafarani
处理文本中的重复字符具有挑战性，因为它们既可以表示单词的正确拼写，也可以表示社交媒体帖子中常见的非正式字符拉长现象。我们提出了 CANDLE，这是一个轻量级的阿拉伯语字符级噪声去重系统，能够在不依赖手工规则、词典或形态分析器的情况下解决这一问题。CANDLE 的核心是对连接时间分类（CTC）方法在该任务上的创新应用，这种公式形式此前尚未用于字符去重，它将规范化建模为基于字符编码器的序列对齐问题。我们在三个基准数据集上进行了评估，涵盖干净的报纸文本、手动整理的模糊案例以及真实世界的社交媒体文本，CTC 模型实现了低至 5.37% 的句子错误率（SER），并且始终大幅优于基于分类的基线模型。为了减少推理开销，我们将 6 层 CTC 模型蒸馏为 2 层学生模型，在性能损失极小的前提下实现了深度降低 3 倍。除了去重准确性之外，规范化还带来了实际的下游好处：在多样化的阿拉伯语大语言模型标记器中，相对减少了高达 12.8% 的标记生成量，从而直接降低了推理成本并提高了上下文窗口的利用率。我们公开了所有代码和模型，以支持可复现性并推动未来的研究。 核心贡献：CANDLE 提出了一种基于 CTC 的轻量级阿拉伯语字符级噪声去重方法，并通过模型压缩和实际下游效益验证了其高效性和实用性。

CN-NewsTTS Bench: a target-level automatic benchmark for raw-input Chinese news TTS pronunciation
👥 Shijun Luo
中文摘要：中文新闻文本中包含大量密集的书面形式，如分数、连字符型号名称、范围、单位符号、百分比、英文缩写以及中英数字混合的名称。这些形式在实际语音流程中频繁出现，而文本到语音（TTS）系统可能会在保留书面字符串的同时改变发音含义。我们引入了CN-NewsTTS Bench v0.1，这是一个开放的目标级基准测试集，用于评估中文新闻TTS产品是否能从原始文本中正确发音这些目标，无需用户端规则、大型语言模型重写、SSML提示或手动编辑。该发布包括一个200条记录的开发集、一个800条记录的公共测试集、992个可自动评估的目标、来自三个ASR模型集成的固定转录本、一个自动目标评分器以及七个产品TTS系统的初步结果。此外，我们还报告了ASR路径诊断、ASR子集消融实验、类别级别结果、置信区间和供应商配置元数据。最佳系统的严格准确率为0.879，而一些系统仍低于0.60。 核心贡献：提出了CN-NewsTTS Bench v0.1，这是首个面向原始输入中文新闻TTS发音评估的目标级自动基准测试集。

DREAM: Dense Retrieval Embeddings via Autoregressive Modeling
👥 Yixuan Tang, Yi Yang
Dense retrieval embedding models are a fundamental component of modern retrieval-based AI systems. Most dense retrievers are trained with contrastive objectives, which require labeled positive and negative document pairs that are often costly and difficult to obtain. In this work, we investigate whether the autoregressive next-token prediction objective of a large language model (LLM) can provide supervision for dense retrieval. The intuition is simple: if a document contains information relevant to a query, conditioning on that document should make the target output easier for the LLM to predict. A key challenge is that the next-token prediction loss is computed inside the LLM, while the retriever is a separate embedding model. To address this challenge, we propose DREAM (Dense Retrieval Embeddings via Autoregressive Modeling), which injects retriever-generated query-document similarity scores into selected attention heads of a frozen LLM. During training, these scores determine how much attention each candidate document receives while the LLM predicts the target output. The resulting prediction loss provides gradients for retriever training through the attention mechanism. We evaluate DREAM on retrieval benchmarks BEIR and RTEB using embedding backbones ranging from 0.5B to 3B parameters. DREAM consistently outperforms existing baselines across different model scales. These results demonstrate that DREAM provides a promising approach for training dense retrievers through autoregressive modeling.
