🔥 大模型前沿

MRI-Eval: A Tiered Benchmark for Evaluating LLM Performance on MRI Physics and GE Scanner Operations Knowledge
👥 Perry E. Radau
背景：现有的MRI大语言模型（LLM）基准主要依赖于复习书籍中的多项选择题，而顶级专有模型在这些题目上已经得分很高，限制了区分度。目前尚无系统性的基准来评估与研究型MRI实践核心相关的特定厂商扫描仪操作知识。目的：我们开发了MRI-Eval，这是一个分层的基准测试工具，用于通过多项选择题（MCQ）对MRI物理和GE扫描仪操作知识进行模型间的相对比较，并结合仅题干和引导性诊断条件作为补充分析。方法：MRI-Eval包含来自教科书、GE扫描仪手册、编程课程材料和专家生成问题的九个类别和三个难度层级中的1365道评分题目。评估了五个模型家族（GPT-5.4、Claude Opus 4.6、Claude Sonnet 4.6、Gemini 2.5 Pro、Llama 3.3 70B）。MCQ是主要评估方式；仅题干测试去除了选项并由独立的LLM判断；引导性题干测试则检验模型对错误用户声明的反应。结果：总体MCQ准确率为93.2%至97.1%。所有模型在GE扫描仪操作类别的表现最低（88.2%至94.6%）。在仅题干条件下，前沿模型的准确率降至58.4%至61.1%，而Llama 3.3 70B降至37.1%；GE扫描仪操作的仅题干准确率为13.8%至29.8%。结论：高MCQ性能可能掩盖自由文本回忆的薄弱，尤其是在特定厂商的操作知识方面。MRI-Eval作为相对比较基准最具信息价值，而非绝对能力衡量，并支持在使用原始LLM输出指导GE特定协议时保持谨慎。 核心贡献：MRI-Eval提供了一个结构化的分层基准，揭示了当前大语言模型在MRI物理和GE扫描仪操作知识上的实际表现差异，特别是在自由文本生成任务中暴露的知识短板。

PSK at SemEval-2026 Task 9: Multilingual Polarization Detection Using Ensemble Gemma Models with Synthetic Data Augmentation
👥 Srikar Kashyap Pulipaka
我们提出了在SemEval-2026任务9：多语言观点两极化检测中的系统，这是一个涵盖22种语言的二分类任务。我们的方法通过低秩适配（LoRA）对每种语言分别微调独立的Gemma~3模型（12B和27B参数），并使用大型语言模型（LLM）生成的合成数据进行增强。我们采用三种合成数据策略（直接生成、改写和对比对创建），利用GPT-4o-mini生成数据，并结合基于嵌入的去重的多阶段质量过滤流程。我们发现，在开发集上针对每种语言进行阈值调整可以在不重新训练的情况下带来2%到4%的F1分数提升。此外，我们还根据每种语言选择策略，对12B和27B模型的预测结果进行加权集成。最终系统在所有22种语言上的平均宏F1得分为0.811，总体排名参赛队伍中的第2名，在3种语言中获得第一名，在8种语言中进入前三名。我们还发现，虽然XLM-RoBERTa和Qwen3等替代架构在开发集上表现强劲，但在测试集上F1分数下降了30%到50%，突显了泛化能力的重要性。 核心贡献：本文提出了一种基于集成Gemma模型与合成数据增强的多语言观点两极化检测方法，在22种语言上取得了优异的性能，并强调了模型泛化的重要性。

Beyond Semantics: An Evidential Reasoning-Aware Multi-View Learning Framework for Trustworthy Mental Health Prediction
👥 Yucheng Ruan, Ling Huang, Qika Lin
使用文本数据进行自动心理健康预测在深度学习和大型语言模型的支持下已经取得了令人鼓舞的成果。然而，将这些模型部署到高风险的现实场景中仍然面临挑战，因为现有方法大多依赖语义表示，并且在面对模糊、嘈杂或分布偏移的数据时常常产生过于自信的预测。此外，大多数方法缺乏可靠的不确定性估计，这削弱了其在高风险心理健康应用中的可信度。为了解决这些问题，我们将任务建模为一个多视角学习问题，结合编码器模型提取的语义信息与解码器模型提取的更高层次推理信息，在可信的方式下获得具有推理感知的表示和不确定性建模。为了确保可靠的融合，我们采用基于主观逻辑的证据学习框架来显式建模不确定性，并引入一种证据融合策略，该策略在平衡互补视角的同时折扣不可靠的证据。在三个真实世界数据集 Dreaddit、SDCNL 和 DepSeverity 上的基准测试分别报告了 0.835、0.731 和 0.751 的准确率，展示了其在可靠心理健康预测方面的潜力。关于噪声鲁棒性和可解释性的附加实验表明，我们提出的框架不仅提升了预测性能，还提供了可信的不确定性估计和人类可理解的推理信号，使其适用于心理健康评估中的高风险应用场景。 核心贡献：本文提出了一种融合语义与推理信息并建模不确定性的多视角学习框架，显著提升了心理健康预测的可靠性与可解释性。

Text Corpora as Concept Fields: Black-Box Hallucination and Novelty Measurement
👥 Nicholas S. Kersting, Vittorio Castelli, Chieh Ting Yeh
我们引入了文本语料库的**概念场（Concept Field）**：一种在句子嵌入空间中从连续句子之间的差异（delta）估计出的局部漂移场，并带有逐点不确定性。对于一个候选句子转换，我们通过 $ζ$ 来评分其与该场的一致性，即观察到的差值与场的局部高斯估计之间的平均绝对z距离。该评分方法是黑盒的（无需模型内部结构）、可追溯至语料库句子的（每个评分都可追溯到附近语料中的句子），并且具有直接的概率解释。我们为此计算提供了一个**向量序列数据库（Vector Sequence Database, VSDB）**，其中存储了嵌入、序列位置和下一个差值元数据。我们在两个大规模场景中评估了这种方法：一是基于美国联邦法规的幻觉式“扎根性”检测，二是基于Project Gutenberg的新颖性检测。使用受控的LLM生成重写内容，概念场在“有依据 / 无依据 / 不确定”的分类策略下表现出色；相比以检索为中心的基线方法，它在两个领域内的覆盖-风险行为相似，支持跨领域的概率解释。我们还展示了如何通过对密集聚类的概念场计算散度和旋度，揭示出定性上有意义的语义模式（如逻辑源、汇和隐含主题），我们将这些视为假设生成工具而非定量结果。概念场为判断句子是否“扎根”或新颖提供了快速、轻量且可解释的信号，可以作为LLM作为裁判者以及白盒检测器的补充。 核心贡献：本文提出了一种名为“概念场”的新方法，用于在不依赖模型内部结构的情况下，对文本的扎根性和新颖性进行高效、可解释的评估，并通过向量序列数据库实现，适用于大规模法律和文学语料分析。

Continual Knowledge Updating in LLM Systems: Learning Through Multi-Timescale Memory Dynamics
👥 Andreas Pattichis, Constantine Dovrolis
LLMs 通常一次性训练后，就被部署到一个不断变化的世界中。外部记忆系统可以弥补这一问题，但大多数系统是显式管理记忆的，而不是让其自行适应。生物记忆则不同：耦合的多时间尺度动态机制使新知识能立即使用、重复确认的知识会被强化，而其余内容则逐渐淡忘。我们认为外部记忆也应遵循类似原则。在 Memini 中，这种观点体现为一种将知识组织成有向图的联想记忆系统。每条边包含两个耦合的内部变量（一个快速，一个缓慢），遵循 Benna-Fusi 模型中的突触巩固机制。通过这种耦合，事件敏感性、逐步巩固和选择性遗忘作为单一机制的不同方面浮现出来，从而将外部记忆重新定义为可通过自身动态重组的学习基质。 核心贡献：提出 Memini 系统，通过模拟生物记忆的多时间尺度动态机制，实现外部记忆的自适应学习与重组，使大语言模型能够持续更新知识。

Automatically Finding and Validating Unexpected Side-Effects of Interventions on Language Models
👥 Quintin Pope, Ajay Hayagreeve Balaji, Jacques Thibodeau
We present an automated, contrastive evaluation pipeline for auditing the behavioral impact of interventions on large language models. Given a base model $M_1$ and an intervention model $M_2$, our method compares their free-form, multi-token generations across aligned prompt contexts and produces human-readable, statistically validated natural-language hypotheses describing how the models differ, along with recurring themes that summarize patterns across validated hypotheses. We evaluate the approach in synthetic setting by injecting known behavioral changes and showing that the pipeline reliably recovers them. We then apply it to three real-world interventions, reasoning distillation, knowledge editing and unlearning, demonstrating that the method surfaces both intended and unexpected behavioral shifts, distinguishes large from subtle interventions, and does not hallucinate differences when effects are absent or misaligned with the prompt bank. Overall, the pipeline provides a statistically grounded and interpretable tool for post-hoc auditing of intervention-induced changes in model behavior.

📊 广告与推荐算法

FEDIN: Frequency-Enhanced Deep Interest Network for Click-Through Rate Prediction
👥 Zenan Dai, Jinpeng Wang, Junwei Pan
摘要：序列推荐模型通常难以捕捉用户兴趣中的潜在周期性模式，这主要是由于时域行为数据中固有的噪声所致。虽然频域分析能够提供全局视角来应对这一问题，但现有方法通常孤立地处理用户序列，忽略了目标物品的重要上下文信息。在本研究中，我们提出了一项新的实证观察：当以正负目标物品为条件时，用户的注意力得分表现出不同的谱熵分布。具体而言，真实用户兴趣在频域中表现为高度集中的谱模式，具有较低的熵值，而无关行为则表现为高熵噪声。基于这一发现，我们提出了频率增强深度兴趣网络（FEDIN）。FEDIN引入了一个频域分支，利用目标感知的频谱过滤机制来分离这些周期性兴趣信号。在三个公开数据集上的大量实验表明，FEDIN始终优于最先进的序列推荐基线模型，并展现出更强的抗噪鲁棒性。我们的代码已发布在：https://github.com/otokoneko/FEDIN。 核心贡献：本文提出了一种新颖的频率增强深度兴趣网络（FEDIN），通过引入目标感知的频谱过滤机制，有效捕捉用户兴趣中的周期性模式，提升了序列推荐的抗噪能力和性能表现。
